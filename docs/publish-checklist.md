# Publish Checklist

Чек перед публикацией каждой страницы калькулятора.

## Контент и структура

- [ ] H1 на месте и соответствует странице калькулятора.
- [ ] Shortcode стоит сразу под H1 (above the fold).
- [ ] Есть блок «Как производится расчет».
- [ ] Есть блок FAQ.
- [ ] Есть таблица норм (СП/ГОСТ).
- [ ] Есть внутренние ссылки на другие калькуляторы.
- [ ] Есть дисклеймер:
  - расчет ориентировочный;
  - на основании действующих СП и ГОСТ.

## Проверка формы

- [ ] Валидный кейс:
  - поля заполнены корректно;
  - форма возвращает результат (объем и количество материала).
- [ ] Невалидный кейс:
  - хотя бы одно обязательное поле не заполнено или <= 0;
  - форма показывает понятную ошибку.

## UX и техническая проверка

- [ ] Мобильная проверка выполнена (ширина 375px в DevTools).
- [ ] На мобильной ширине форма не ломается, поля и кнопка удобны для нажатия.
- [ ] Нет битых внутренних ссылок на странице.

## Финальная публикация

- [ ] Страница опубликована.
- [ ] Страница добавлена в нужные разделы меню/перелинковки.
