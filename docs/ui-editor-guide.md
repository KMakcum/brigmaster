# UI Editor Guide

## Для кого эта инструкция

Этот файл для контент-редактора WordPress (без программирования), который добавляет калькуляторы на страницы сайта.

## Как добавить калькулятор в админке WordPress

1. Войдите в админку WordPress.
2. Откройте **Страницы -> Добавить новую** (или выберите существующую страницу).
3. Вставьте shortcode в текст страницы.
4. Нажмите **Опубликовать** или **Обновить**.
5. Откройте страницу на сайте и проверьте форму.

## Какой shortcode где использовать

Атрибут `title` задаёт заголовок над формой калькулятора. Если `title` не указан или пустой, заголовок внутри виджета не показывается.

- Для страницы про бетон: `[brigmaster_concrete_estimator title="Калькулятор плитного фундамента"]`
- Для страницы про ленточный фундамент: `[brigmaster_strip_foundation_estimator title="Калькулятор ленточного фундамента"]`
- Для страницы про свайный фундамент: `[brigmaster_pile_foundation_estimator title="Калькулятор свайного фундамента"]`
- Для страницы про кирпич: `[brigmaster_brick_estimator title="Калькулятор кирпича"]`
- Для страницы про стяжку: `[brigmaster_screed_estimator title="Калькулятор стяжки"]`
- Для страницы про гипсокартон: `[brigmaster_drywall_estimator title="Калькулятор гипсокартона"]`
- Для страницы про плитку: `[brigmaster_tile_estimator title="Калькулятор плитки"]`

Можно размещать по одному калькулятору на отдельной странице или несколько на одной странице.

Для свайного фундамента: у буронабивных свай можно отдельно включить расчет арматуры свай.

## Что объяснять пользователю на странице

- **Режим расчета** - вариант расчета (норматив, с запасом, для новичка).
- **Площадь** - вводится в м².
- **Толщина** - вводится в метрах. Пример: `0.1` = `10 см`.
- После нажатия **Рассчитать** пользователь видит:
  - **Объем**
  - **Количество материала**

## Типичные ошибки ввода и как их объяснять

- Ошибка: не выбран режим.
  - Как объяснить: "Выберите режим расчета перед отправкой формы."
- Ошибка: площадь меньше или равна нулю.
  - Как объяснить: "Площадь должна быть больше нуля."
- Ошибка: толщина меньше или равна нулю.
  - Как объяснить: "Толщина должна быть больше нуля и вводится в метрах."
- Ошибка сети (например, пропал интернет).
  - Как объяснить: "Проверьте подключение к интернету и попробуйте еще раз."

## Мини-чек перед публикацией страницы

1. Shortcode вставлен без опечаток.
2. Форма отображается на странице.
3. Валидный ввод показывает результат.
4. Невалидный ввод показывает понятные ошибки.
